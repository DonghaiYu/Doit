# Doit 免安装版使用指南

## 🎉 欢迎使用 Doit 任务管理器 (免安装版)

这是一个**无需安装、解压即用**的版本,专门为没有 Apple 开发者签名的应用设计。

---

## 📖 三步快速开始

### 第一步: 解压文件
1. 下载 `Doit-x.x.x-portable-mac.zip`
2. 双击解压到任意位置(如桌面、文档文件夹等)
3. 解压后会得到一个同名文件夹

### 第二步: 启动应用
**方法 A - 推荐方式 (自动处理所有问题):**
- 双击 `launch-mac.sh` 文件
- ✅ 自动处理 macOS 安全限制
- ✅ 自动启动应用

**方法 B - 直接启动:**
- 双击 `Doit.app` 图标
- 如果提示"已损坏"或"无法验证开发者",请右键点击 → 选择"打开"

### 第三步: 享受使用
- 应用启动后可以正常使用所有功能
- 所有数据保存在本地,完全隐私安全

## 🛠️ 常见问题解答

### Q1: 双击 launch-mac.sh 没反应?
**A:** 可能是脚本没有执行权限,请尝试:
1. 打开"终端"应用
2. 输入: `cd` (注意 cd 后面有空格)
3. 将解压后的文件夹拖入终端窗口
4. 按回车
5. 输入: `bash launch-mac.sh`

### Q2: 仍然提示"已损坏"?
**A:** 使用 `fix-quarantine.sh` 脚本:
1. 双击 `fix-quarantine.sh` 文件
2. 或在终端运行: `bash fix-quarantine.sh`

### Q3: 应用无法启动?
**A:** 检查以下几点:
1. 确认是 macOS 系统(不支持 Windows/Linux)
2. 确认是 Apple Silicon (M1/M2/M3) 或 Intel 处理器
3. 查看错误日志: `~/Library/Logs/Doit/`

### Q4: 数据保存在哪里?
**A:** 数据保存在应用内部,位置如下:
- 打开应用后点击"设置" → "数据导出"
- 可以导出所有数据为 JSON 格式备份

### Q5: 可以升级到正式版吗?
**A:** 可以!下载正式版安装程序:
- 正式版会自动升级
- 数据会自动迁移

## 💡 使用技巧

### 1. 创建快捷方式
如果想快速访问,可以:
- 右键点击 `launch-mac.sh` → 选择"制作别名"
- 将别名放到桌面

### 2. 固定到 Dock
- 应用启动后,在 Dock 中右键点击图标
- 选择"选项" → "保留在 Dock 中"

### 3. 定期备份数据
- 定期使用"数据导出"功能
- 将导出的 JSON 文件保存到安全位置
