## 📝 详细步骤

### 1️⃣ 下载文件
```
Doit-1.3.0-portable-mac.zip (90.21 MB)
```

### 2️⃣ 解压文件
- 双击 zip 文件解压
- 解压后文件夹内容:
  ```
  Doit-1.3.0-portable-mac/
  ├── Doit.app          ← 应用程序
  ├── launch-mac.sh     ← 启动脚本(双击这个!)
  ├── fix-quarantine.sh ← 修复脚本
  ├── README.md
  └── 使用说明.md
  ```

### 3️⃣ 启动应用
**推荐方式:**
- 双击 `launch-mac.sh`
- ✅ 自动处理安全限制
- ✅ 自动启动应用

**备用方式:**
- 双击 `Doit.app`
- 如果被阻止，右键 → "打开"

## 🔧 问题快速修复

### ❌ "无法验证开发者" 或 "已损坏"
**方案 A (推荐):** 双击 `launch-mac.sh`

**方案 B:** 双击 `fix-quarantine.sh`

**方案 C:** 右键 `Doit.app` → "打开"

### ❌ 双击脚本没反应
打开终端执行:
```bash
cd [拖入解压后的文件夹路径]
bash launch-mac.sh
```

### ❌ 提示"未找到 Doit.app"
确保 `launch-mac.sh` 和 `Doit.app` 在同一目录下

## 📊 版本选择建议

### 选择免安装版，如果:
- ✅ 没有 Apple 开发者账号
- ✅ 需要快速分享给其他人
- ✅ 不想安装到系统
- ✅ 需要从 U 盘运行
- ✅ 频繁在不同电脑间使用

### 选择 DMG 正式版，如果:
- ✅ 希望像普通应用一样安装
- ✅ 有 Apple 开发者签名
- ✅ 需要长期使用
- ✅ 希望固定到 Dock

### 选择 ZIP 版本，如果:
- ✅ 需要手动拖入应用文件夹
- ✅ 需要完全控制安装位置

## 🛡️ 安全说明

### 为什么免安装版安全?
1. **开源透明**: 代码完全公开，可审查
2. **无网络请求**: 完全离线运行
3. **本地存储**: 数据保存在本地
4. **无追踪**: 不收集任何用户信息

### macOS 的保护机制
- Gatekeeper 会阻止未签名应用
- 但这是**正常的安全机制**
- 使用启动脚本可以自动处理
- 不影响应用功能和安全性
