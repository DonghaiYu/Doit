# Doit 安装说明

## macOS 安装指南

由于本应用未经过 Apple 公证,首次安装时需要进行以下操作:

### 方法一:右键打开(推荐)

1. 将 `Doit.app` 拖入"应用程序"文件夹
2. **不要**直接双击打开
3. 右键点击 `Doit.app`,选择"打开"
4. 在弹出的对话框中点击"打开"按钮
5. 之后就可以正常使用了

### 方法二:系统设置

1. 尝试打开应用(会被 macOS 阻止)
2. 打开"系统设置" → "隐私与安全性"
3. 在底部找到关于 Doit 的提示
4. 点击"仍要打开"按钮
5. 在弹出的对话框中确认"打开"

### 方法三:终端命令(如果提示"已损坏")

如果上述方法仍然无法打开,在终端中运行:

```bash
xattr -cr /Applications/Doit.app
```

然后再尝试打开应用。

## Windows 安装指南

Windows 版本可以直接安装使用,可能会有 SmartScreen 警告:

1. 运行安装程序
2. 如果出现"Windows 已保护你的电脑"提示
3. 点击"更多信息"
4. 点击"仍要运行"

## 为什么会有这些提示?

这是因为应用没有经过 Apple 或 Microsoft 的代码签名认证。代码签名需要:
- Apple: 每年 $99 的开发者账号
- Microsoft: 购买代码签名证书(约 $200-500/年)

本应用是开源免费的,因此没有进行付费签名。这不影响应用的安全性和功能。

## 常见问题

**Q: 这个应用安全吗?**  
A: 是的。你可以查看源代码确认应用的安全性。未签名只是因为没有付费购买证书。

**Q: 每次打开都需要右键吗?**  
A: 不需要。只有第一次打开需要,之后可以正常双击打开。

**Q: 可以信任这个应用吗?**  
A: 可以。应用是开源的,你可以自己编译或查看源代码。
